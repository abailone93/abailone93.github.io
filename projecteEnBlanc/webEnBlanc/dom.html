DOM (Document Object Model)

DOM Style

El style representa un estat d'estil individual. Per exemple: backgroundColor, border, font, opacity, etc.

Sintaxis:

object.style.font = "font-style font-variant font-weight font-size/line-height|caption|icon|menu| message-box|small-caption|status-bar|initial|inherit;"

object.style.border = "width style color|initial|inherit"

object.style.backgroundColor = "color|transparent|initial|inherit"

object.style.opacity = "number|initial|inherit"

DOM Element

DOM Element representa el elements com aquest (p) o table, div, a, etc. Dins del que és JavaScript tenim funcions com el click(), getElementsByClassName(), toString(), etc.

Sintaxis:

HTMLElementObject.click()

element.getElementsByClassName(classname)

DOM Document

El Document és un objecte dins la ruta admin del HTML. Tenim varis exemples: addEventListener(), createElement(), getElementById(), hasFocus(), write(), etc.

Sintaxis:

document.write(exp1, exp2, exp3, ...)

document.hasFocus()

document.getElementById(elementID)

document.createElement(nodename)

document.addEventListener(event, function, useCapture)

DOM Events

JavaScript registra diferents events en els elements del document HTML, normalment son utilitzats amb combinacio de funcions. Alguns exemples son: input, mouseover, timeupdate, waiting, load, etc.

Sintaxis:

object.onload = function(){myScript};

object.oninput = function(){myScript};

object.addEventListener("mouseover", myScript);

object.addEventListener("timeupdate", myScript);

object.addEventListener("waiting", myScript);
